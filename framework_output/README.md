# framework_output

本目录存放方法论提取框架的主交付物。

## 目录结构

- `main/`：框架正文（`方法论提取框架_v0.1_trial.md` + `.json`）
- `components/`：组件模块（占位）
- `schemas/`：JSON schema、证据卡模板（占位）
- `examples/`：示例证据卡、示例审查输出（占位）
